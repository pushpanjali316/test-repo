## Getting Started

To get started with the notebooks in this repository:

1. Clone the repo:
   ```bash
   git clone https://github.com/pushpanjali316/test-repo.git
   ```
2. Change into the project directory:
   ```bash
   cd test-repo
   ```
3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
4. Open any `.ipynb` file to begin exploring!

## Requirements

- [Python 3.x](https://www.python.org/)
- [Jupyter Notebook](https://jupyter.org/)
- Additional Python dependencies (see individual notebook cells for required libraries)

You can install Jupyter Notebook using pip:
```bash
pip install notebook
```
